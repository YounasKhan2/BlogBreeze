## What Each Component Does

### GitHub (Code Storage)
- Stores your code
- Tracks changes
- Triggers deployments

### Render (Hosting Platform)
- Runs your application
- Manages database
- Handles HTTPS/SSL
- Provides URL

### PostgreSQL (Database)
- Stores blog posts
- Stores user accounts
- Stores comments
- Stores categories/tags

### Gunicorn (Web Server)
- Serves your Django app
- Handles HTTP requests
- Manages connections

### WhiteNoise (Static Files)
- Serves CSS/JS/images
- Compresses files
- Caches efficiently
